# Installation Guide

## Backend Installation
```sh
cd project
cd backend
npm install
npm run dev
http://localhost:5000/api/docs
```

![Alt Text](https://drive.google.com/uc?export=view&id=1Hlpe2dimtK3H_NDJeKEup5IMnOpliCPL)

## Frontend Installation
```sh
cd project
cd frontend
npm install
npm run dev
http://localhost:3000
```
